⭐ 별 표시 테스트

별이 보이시나요?

테스트 중인 별들:

  • 작은 흰색 별: 10개 (body::before)
  • 큰 보라/파란 별: 5개 (body::after)
  • 크기: 3-5px (매우 크게 설정)
  • 불투명도: 1 (완전 불투명)
  • z-index: 0 (배경 레이어)

확인 방법:

  1. 이 검은 배경에 흰색/보라색 점들이 보여야 합니다
  2. 브라우저 개발자 도구 (F12) → Elements 탭 열기
  3. body 태그 선택 → Styles 패널에서 ::before와 ::after 확인
  4. box-shadow 속성이 적용되어 있는지 확인

❓ 별이 안 보인다면:

  • 브라우저 캐시 문제일 수 있습니다 (Ctrl+Shift+R로 새로고침)
  • CSS 파일이 제대로 로드되지 않았을 수 있습니다
  • z-index 문제로 다른 요소에 가려졌을 수 있습니다
