# PDF Files Folder

이 폴더는 발표 슬라이드, 포스터, 논문 등의 PDF 파일을 저장하는 곳입니다.

## 📁 파일 추가 방법

1. PDF 파일을 이 폴더에 복사
2. `data/publications.json` 또는 `data/news.json`에 링크 추가
3. Git 커밋 및 푸시

## 🔗 링크 형식

```json
{
  "links": {
    "slides": "assets/pdf/your_file.pdf"
  }
}
```
